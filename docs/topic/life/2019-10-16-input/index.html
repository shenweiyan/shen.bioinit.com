输入法

2019-10-16

现在移动端电脑端的输入法有很多。但有一个问题：安卓手机上哪个输入法可以左右滑动键盘来移动光标？

我用的是华为手机，目前搜狗输入法是可以的。

第二个问题，左右角引号（「」）怎么打在搜狗输入法中怎么打出来？ctrl+shift+z 快捷键弹出搜狗输入法特殊符号，在 “标点符号” 中选择。
