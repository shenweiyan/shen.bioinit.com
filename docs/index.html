愿上帝赐予我平静，让我能接纳我无法改变的事；

愿上帝赐予我勇气，让我能改变我可以改变的事；

用心生活每一天；用灵魂享受每个时刻；承受磨难，因为它是通向安宁的必经之路。

这样，这一生我就有理由得到快乐，并在天堂与您一起得到极乐。
