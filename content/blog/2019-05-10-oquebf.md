1. 本地创建分支 v-1.2
```shell
$ git branch v-1.2
$ git branch
* master
  v-1.2
```

2. 切换本地分支
```shell
$ git branch
* master
  v-1.2

$ git checkout v-1.2
Switched to branch 'v-1.2'

$ git branch
  master
* v-1.2
```

3. 把本地分支提交到远程分支仓库
```shell
$ git add --all

$ git commit -m "ASO design for Galaxy version 1.2"

$ git push origin v-1.2
```

4. 查看一下远程仓库有几个分支
```shell
$ git branch -a
  master
* v-1.2
  remotes/origin/HEAD -> origin/master
  remotes/origin/master
  remotes/origin/v-1.2
```
